# CI/CD Failure Investigator

An AI-powered platform that investigates CI/CD build failures and proposes fixes
using a LangGraph agent, RAG over past failures, and a terminal-style Next.js dashboard.

## Stack
- Backend: FastAPI, LangGraph, LangChain, SQLAlchemy
- Databases: MySQL, ChromaDB
- Frontend: Next.js 14, React, Tailwind CSS
- Deploy: Docker Compose, Railway
